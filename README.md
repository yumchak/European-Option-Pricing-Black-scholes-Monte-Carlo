# Option-pricing-and-Sensitivity-Analysis
This project implements the Black–Scholes model and Monte Carlo simulation
to analyse the pricing of European call options. The project compares
analytical pricing with numerical simulation and studies the sensitivity
of option prices to key parameters such as volatility and time to maturity.

The stock price dynamics are assumed to follow Geometric Brownian Motion
(GBM). Python is used to simulate stock price paths, compute option prices,
and visualise the results.
